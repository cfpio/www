Features

Free and Open

As event organizers we developped a Call for Papers application. We decided to make it available to everybody as a free as-a-service offer.
CFP.io is opensource, and we welcome contributions if you miss some feature or discover a bug.

We're non-profit organization, so won't ask you to pay for it and will respect your data privacy. We just ask for a free pass to your event so we can join or resell the pass to pay for travel.
If your event is free, that's ok as well. We will just suggest to drink a beer together if we can meet face to face.

Describe your event

Define your event details and customize Call for Papers accordingly. Define event topic using themes, speakers will select when they submit a proposal. Define session formats and duration with help tips to assist speaker in selecting the ones that will best match their proposals.

Review proposals

Review proposals and interact with speaker using message system, as well in private mode with the organizer team. Eventually vote for each proposals as other teammates and establish CFP ranking.

Better visibility

Speakers already using CFP.io for another event will easily discover yours. Thanks to event keywords we can suggest them other events they could be interested in. As a result, you'll get more and better proposals.

More features

All event organizers have very comparable requirement, with subtle variations. Compared to other solutions, CFP.io is open-source hosted solution. This means you are welcome to contribute, or at least request features or third party integrations you're missing (report them here).

Schedule and publish

CFP.io do support event scheduling and integration with various event agenda services. Based on proposal review ranking, establish event schedule and assign all selected talks a slot in your agenda. Publish event and ensure it get up-to-date as you need to make some last minute changes to workaround speakers' logistic constraints.

Integration

Your CFP data is available with all the required APIs so you can integration with third party services or your own tools. Missing something ? Just ask.
